#### 정규표현식을 통한 데이터 변환 
/23232 형식의 문자를 공백으로 전환

```json
 mutate {
    gsub => [
        "url","\/(?<![0-9.+-])(?>[+-]?(?:(?:[0-9]+(?:\.[0-9]+)?)|(?:\.[0-9]+)))",""
        ]
}
```
#### JSON 구조
json 구조의 data를 es에 넣을 때 
parsedjson 필드에 자동으로 들어간다. 
```json 
json {
    source => "json"
    target => "parsedjson"
}
```

##### QueryString을 key/value 넣을 때

```json 
 kv {
            source       => "json"
            field_split  => "&?"
    }
```

#### 필드를 복사 
source 필드를 source_tmp 하나 더 생성 

```json 
    mutate {
        copy => { "[source]" => "source_tmp" }
      }
```
